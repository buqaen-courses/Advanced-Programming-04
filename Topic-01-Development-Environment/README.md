# Subject 1: Git Basics & Development Environment Setup

## Overview

This subject provides a comprehensive foundation in version control and development environment setup. Students will master Git fundamentals, learn to configure professional development environments across different platforms, and understand essential collaboration workflows. The focus is on practical skills that form the backbone of modern software development.

## Learning Path

### 1. Development Environment Setup
- Installing Git across different platforms (Windows, macOS, Linux)
- Configuring development tools and editors
- Setting up terminals and command-line environments
- Understanding development toolchains

### 2. Git Fundamentals
- Understanding version control concepts
- Repository creation and management
- Basic Git operations (init, add, commit, status)
- Working with local repositories

### 3. Advanced Git Operations
- Understanding Git states (working directory, staging, repository)
- File tracking and management
- Branching and merging concepts
- Collaboration workflows and best practices

## Key Topics

- **Version Control**: Git concepts and importance
- **Repository Management**: Creating, cloning, and maintaining repositories
- **File Operations**: Add, commit, status, and log commands
- **Git States**: Working directory, staging area, and repository
- **Development Tools**: Editors, terminals, and extensions
- **Collaboration**: Branching, merging, and team workflows

## Prerequisites

- Basic computer literacy
- Familiarity with file systems and folders
- Understanding of basic command-line concepts
- Text editor for code editing

## Learning Outcomes

By the end of this subject, students will be able to:

1. **Environment Setup**: Install and configure Git on any major operating system
2. **Tool Proficiency**: Set up and use professional development tools effectively
3. **Git Fundamentals**: Understand version control concepts and Git's role in development
4. **Repository Management**: Create, clone, and manage Git repositories
5. **File Operations**: Use Git commands to track, stage, and commit changes
6. **State Management**: Understand and manage Git's working directory, staging area, and repository
7. **Collaboration Ready**: Prepare for team development workflows and best practices
8. **Problem Solving**: Troubleshoot common Git issues and development environment problems

## Special Focus: Professional Development Environment

This subject emphasizes the importance of a well-configured development environment:

- **Cross-Platform Setup**: Installation guides for Windows, macOS, and Linux
- **Tool Integration**: Understanding how different tools work together
- **Best Practices**: Professional development environment configuration
- **Troubleshooting**: Common setup issues and their solutions

## Progression

1. **Start with Setup** (Installation Guides) - Get your environment ready
2. **Learn Git Basics** (Section 1) - Understand version control fundamentals
3. **Master Repository Operations** (Section 2) - Learn repository management
4. **Deepen Git Knowledge** (Section 3) - Understand Git states and advanced operations
5. **Apply Knowledge** (Homework) - Build real projects with Git
6. **Collaborate Effectively** (Future Topics) - Build on this foundation
