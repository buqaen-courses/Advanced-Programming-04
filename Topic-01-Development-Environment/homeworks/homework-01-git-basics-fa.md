# تمرین اول: مبانی Git و مدیریت مخزن

## اهداف یادگیری
با تکمیل این تمرین، شما قادر خواهید بود:
- مخازن محلی Git را ایجاد و مدیریت کنید
- حالت‌های کاری Git، ناحیه مرحله‌بندی و مخزن را درک کنید
- از دستورات پایه Git استفاده کنید (init, add, commit, status, log)
- ساختارهای پروژه سازمان‌یافته با چندین پوشه ایجاد کنید
- فایل‌ها و پوشه‌ها را از طریق چرخه حیات Git مدیریت کنید

## پیش‌نیازها
- Git روی سیستم شما نصب شده باشد
- درک پایه‌ای از عملیات خط فرمان
- ویرایشگر متن برای ایجاد فایل‌ها

### قسمت ۱: تنظیم پروژه و مقداردهی اولیه Git

۱. **یک پوشه جدید برای پروژه تمرین خود ایجاد کنید:**
   ```bash
   # یک پوشه به نام شماره دانشجویی خود ایجاد کنید

   ```

## عیب‌یابی

### مشکلات رایج:
۱. **"fatal: not a git repository"**: اطمینان حاصل کنید که در دایرکتوری صحیح هستید و `git init` را اجرا کرده‌اید
۲. **"nothing to commit"**: `git status` را بررسی کنید تا ببینید چه فایل‌هایی نیاز به افزودن دارند
۳. **خطاهای نحوی Python**: فایل‌های Python خود را قبل از کامیت تست کنید

### دستورات مفید:
```bash
# وضعیت فعلی را بررسی کنید
git status

# تاریخچه کامیت را ببینید
git log --oneline

# ببینید چه تغییری در هر کامیت ایجاد شده
git show <commit-hash>

# فایل‌ها را unstaged کنید اگر نیاز باشد
git reset HEAD <file>
```
